## ✨ Features

- 🌍 World Financial Galaxy
- 💱 Live Forex Rates
- ₿ Live Crypto Prices
- 🥇 Live Gold Price (XAU/USD)
- 📈 TradingView Charts
- 🔄 Currency Converter
- 🌙 Dark Luxury UI
- 📱 Fully Responsive

## 🚀 Tech Stack

- Next.js 15
- TypeScript
- JavaScript
- Tailwind CSS
- React Three Fiber
- Three.js
- Framer Motion
- GSAP

## 📦 Installation

```bash
git clone https://github.com/kamunanay/cukimai.git
cd cukimai
npm install
npm run dev
```
