## Features

- Real-time Forex
- Real-time Cryptocurrency
- Live Gold Price (XAU/USD)
- Interactive 3D Coin Galaxy
- USD Currency Converter
- TradingView Charts
- Responsive Design
- Dark Premium UI

- JavaScript
- Tailwind CSS

- React Three Fiber

## Installation

```bash
git clone https://github.com/kamunanay/cukimai.git

cd cukimai

npm install

npm run dev
```

---

## Deploy

```bash
npm run build

npm start
```

Compatible with:

- Cloudflare Pages
- Vercel
- Ubuntu VPS
- Debian VPS
